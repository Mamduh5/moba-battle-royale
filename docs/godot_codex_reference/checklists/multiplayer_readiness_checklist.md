# Multiplayer Readiness Checklist

- [ ] Server can run headless.
- [ ] Client/server protocol version check exists.
- [ ] Content manifest check exists.
- [ ] Match token validation exists.
- [ ] Input sequence/ack exists.
- [ ] Movement validation exists.
- [ ] Ability validation exists.
- [ ] Snapshot delivery exists.
- [ ] Correction/reconciliation path exists.
- [ ] Disconnect/reconnect policy exists.
- [ ] Bot fill policy exists.
- [ ] Match result submission path exists.
- [ ] Replay/event logs exist.
- [ ] Invalid client messages are rejected and logged.
